## 개요
`linux`의 `pipe`를 이용하여
`bash`의 일부 기능을 구현한 프로젝트 입니다

<br/>

## 사용법 

cmake tool 설치 필요
```bash
sudo apt install cmake
```

<br/>

### mandtory
: 기본적인 shell 명령어 사용 가능   

```bash
make
```

<br/>
   
### bonus
: &&, ||, () 사용 가능

```
make bonus
```
