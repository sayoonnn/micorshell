# 사용법 

cmake tool 설치 필요
```bash
sudo apt install cmake
```

<br/>

### mandtory
: 기본적인 shell 명령어 사용 가능   

```bash
make
```

<br/>
   
### bonus
: &&, ||, () 사용 가능

```
make bonus
```

# 기타

`linux`의 `pipe`를 이용한 shell script를 만든 프로젝트

※ 부족한 기능이 더 많습니다
